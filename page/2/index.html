【giscus 配置】给博客增加评论功能

Created2024-02-28Updated2024-02-28

giscus是一款开源的github app, 可以用来对repo或者博客等进行评论。我使用的方法是在hexo博客中增加评论功能，具体步骤可见下文。

giscus官网地址

目前文档里有一些配置信息不是很清晰，这里进行详细的配置解释。

安装

点击链接https://github.com/apps/giscus安装giscus app。选择对应的repo。这个repo一定要包含后文用到的repo

获取repo和配置信息

可以点击github中的graphQL链接生成自己的repo_id和category_id。

将 userName 换成具体的 Github 用户名；

将 repoName 换成保存评论数据的 Repo 名称。

hexo中添加giscus配置

修改repo, repo-id, category, category-id

这里踩了一些坑。repo应该是userName/repoName, 一开始只输repoName一直显示没找到
